# Prime Fibernet React Native App

This project is a React Native application built using Expo, TypeScript, and Redux. It serves as a mobile application for Prime Fibernet, providing users with a seamless experience.

```
├── app.json

├── package.json

└── README.md
```

## Getting Started

To get started with the project, follow these steps:

1. **Clone the repository:**
   ```
   git clone <repository-url>
   ```

2. **Navigate to the project directory:**
   ```
   cd Prime_fibernet
   ```

3. **Install dependencies:**
   ```
   npm install
   ```

4. **Run the application:**
   ```
   npm start
   ```

## Features

- Built with React Native and Expo for a smooth mobile experience.
- TypeScript for type safety and better development experience.
- Redux for state management, ensuring a predictable state container.
